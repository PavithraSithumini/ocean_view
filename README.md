Ocean View Resort is a web-based hotel reservation management system developed using Java Servlets, JSP, and MySQL. The system allows hotel staff to manage guest reservations, room availability, and billing from a centralized admin dashboard. Key features include adding and viewing reservations, dynamic room selection with real-time pricing, automated bill generation with guest detail auto-fill, and printable bill receipts. The application follows a layered MVC architecture with separate Controller, Service, DAO, and Model layers, ensuring clean code structure and maintainability. The database integration is handled through JDBC with a centralized DBConnection utility, and the front end is styled with responsive CSS and vanilla JavaScript for a smooth user experience.
